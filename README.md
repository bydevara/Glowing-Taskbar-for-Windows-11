# Glow-Taskbar-for-Windows-11
Glow Taskbar is a theme for Windhawk that adds a glowing effect to your Windows 11 taskbar.
How to install it
Copy the YAML in https://github.com/bydevara/Glow-Taskbar-for-Windows-11/blob/main/Glow%20Taskbar%20with%20custom%20icons.yaml and paste it to Windhawk Windows 11 Taskbar Styler mod in textual mode, and then click "Save Settings", done.
